# 数学概念系统使用指南

## 🎯 系统概述

数学概念系统是游戏的核心教育功能，将抽象的数学理论转化为可交互的学习体验。系统包含16个核心数学概念，涵盖从基础算术到高等分析的完整学习路径。

## 🚀 如何开始

### 1. 进入概念探索
- 在游戏主界面点击右侧的"概念"按钮
- 打开概念选择面板，浏览所有可用概念

### 2. 选择概念
- 概念按难度分为7个等级（★-★★★★★★★）
- 颜色编码：绿色（简单）→ 黄色（中等）→ 红色（困难）→ 紫色（极难）
- 建议按顺序学习，从简单概念开始

### 3. 探索学习
- 点击概念卡片进入详细探索界面
- 阅读概念的直觉解释和历史背景
- 参与交互式挑战来深化理解

## 🎮 交互式挑战类型

### 构造类挑战
- **目标**：按步骤构造数学对象
- **技能**：逻辑思维、系统性思考
- **示例**：皮亚诺公理的自然数构造

### 证明类挑战
- **目标**：完成严格的数学证明
- **技能**：逻辑推理、严密思维
- **示例**：√2无理性的反证法证明

### 可视化挑战
- **目标**：通过图形理解抽象概念
- **技能**：几何直觉、空间想象
- **示例**：阶梯悖论的动画演示

### 计算类挑战
- **目标**：应用公式解决实际问题
- **技能**：计算能力、应用思维
- **示例**：三角函数值的精确计算

### 游戏化挑战
- **目标**：在游戏中理解数学原理
- **技能**：策略思维、规律发现
- **示例**：ε-δ定义的博弈游戏

## 📈 学习进度系统

### 进度跟踪
- 每个概念的掌握进度：0-100%
- 完成挑战获得进度增益：15-45%
- 不同挑战类型有不同的进度价值

### 掌握奖励
- 概念完全掌握后获得修为奖励：50-450经验
- 高难度概念给予更多奖励
- 触发成就系统和境界提升

### 学习路径
- 建议按难度递增顺序学习
- 某些概念有前置要求
- 系统会推荐适合当前水平的概念
