# 快速开始指南

## 5 分钟快速体验

### 1. 安装依赖

```bash
npm install
```

### 2. 启动本地开发服务器

```bash
npm run dev
```

游戏将在 `http://localhost:8788` 启动

### 3. 开始游戏

1. 打开浏览器访问 `http://localhost:8788`
2. 点击"开始游戏"
3. 使用方向键或 WASD 移动角色
4. 点击橙色的"数学之灵"解答题目
5. 点击绿色的资源点收集材料
6. 点击右上角"合成"按钮制作丹药

## 游戏操作

### 移动控制
- **方向键** 或 **WASD** - 移动角色

### 游戏界面
- **数学之灵** (橙色圆圈) - 点击解答数学题目
- **资源点** (绿色/青色方块) - 点击收集材料
- **保存** - 保存游戏进度
- **背包** - 查看收集的物品
- **合成** - 使用材料合成丹药

### 数学挑战
- 点击数学之灵后会出现题目
- 选择正确答案获得修为
- 答错可以重试
- 修为达到要求后自动提升境界

## 游戏目标

1. **提升境界** - 从炼气期开始，逐步提升至渡劫期
2. **探索区域** - 解锁新区域需要达到相应境界
3. **收集资源** - 收集灵草和矿石
4. **合成物品** - 使用材料合成有用的丹药

## 境界等级

- 炼气 (0 修为)
- 筑基 (100 修为)
- 金丹 (300 修为)
- 元婴 (700 修为)
- 化神 (1500 修为)
- 炼虚 (3000 修为)
- 合体 (6000 修为)
- 大乘 (12000 修为)
- 渡劫 (25000 修为)

## 区域解锁

- **青石村** - 初始解锁，对应炼气境
- **五行山** - 需要金丹境
- **上古遗迹** - 需要元婴境

## 提示

1. 答对题目可以获得修为，答错可以重试
2. 收集材料后可以在合成界面制作丹药
3. 修为丹可以快速提升修为
4. 记得经常保存游戏进度
5. 提升境界后会自动解锁新区域

## 故障排除

### 游戏无法加载
- 检查浏览器控制台是否有错误
- 确认已运行 `npm install`
- 尝试清除浏览器缓存

### 保存功能不工作
- 本地开发时，如果没有配置 KV Storage，数据会保存在内存中
- 部署到 Cloudflare Pages 后需要配置 KV Storage（见 DEPLOY.md）

### 题目不显示
- 检查浏览器控制台错误
- 确认 Phaser.js 已正确加载
