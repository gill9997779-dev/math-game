# 地图背景图片说明

## 📁 图片放置位置

所有地图背景图片应放在以下目录：

```
assets/images/zones/
```

## 📝 命名规则

每个地图的背景图片必须按照以下命名规则：

| 地图名称 | 文件名 | 完整路径 |
|---------|--------|---------|
| 青石村 | `青石村_background.png` | `assets/images/zones/青石村_background.png` |
| 五行山 | `五行山_background.png` | `assets/images/zones/五行山_background.png` |
| 上古遗迹 | `上古遗迹_background.png` | `assets/images/zones/上古遗迹_background.png` |
| 天机阁 | `天机阁_background.png` | `assets/images/zones/天机阁_background.png` |

## 🎨 图片要求

### 尺寸建议
- **推荐尺寸**: 1920x1080 (16:9) 或更高分辨率
- **最小尺寸**: 1280x720 (16:9)
- **格式**: PNG、JPG 或 WebP

### 设计建议
- **青石村**: 宁静的村庄场景，绿色调，适合新手区
- **五行山**: 山峰环绕的神秘之地，棕色调，适合进阶区
- **上古遗迹**: 古老的遗迹场景，深紫色调，适合高阶区
- **天机阁**: 神秘的数理秘境，深紫色调，适合进阶区

## 🔄 后备机制

如果某个地图的背景图片不存在，游戏会按以下顺序尝试：

1. **地图专属背景** (`地图名称_background.png`)
2. **通用背景** (`assets/images/game_background.png`)
3. **渐变背景 + 区域颜色遮罩** (如果以上都不存在)

## 📂 目录结构示例

```
assets/
└── images/
    ├── zones/
    │   ├── 青石村_background.png
    │   ├── 五行山_background.png
    │   ├── 上古遗迹_background.png
    │   └── 天机阁_background.png
    ├── game_background.png (通用背景，可选)
    └── loading_background.png (加载场景背景)
```

## ⚠️ 注意事项

1. **文件名必须完全匹配**：包括中文字符和 `_background.png` 后缀
2. **大小写敏感**：确保文件名的大小写与地图名称完全一致
3. **文件格式**：支持 PNG、JPG、WebP 格式
4. **如果图片不存在**：游戏会使用渐变背景，不会报错，但会在控制台显示警告

## 🎮 如何添加新地图的背景

如果将来添加了新地图，需要：

1. 在 `assets/images/zones/` 目录下添加对应的背景图片
2. 按照命名规则：`地图名称_background.png`
3. 在 `src/scenes/PreloadScene.js` 的 `zoneBackgrounds` 数组中添加新地图名称
4. 重新加载游戏即可
